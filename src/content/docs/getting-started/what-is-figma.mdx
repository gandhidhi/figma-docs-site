Figmaは、ブラウザ上で動作するクラウドベースのUIデザインツールです。インストール不要でチームメンバーとリアルタイムに共同作業できる点が最大の特徴です。

## Figmaが選ばれる理由

<CardGrid>
  <Card title="クラウドネイティブ" icon="cloud">
    ファイルはクラウドに自動保存されます。バージョン管理も自動で行われるため、「上書き保存し忘れた」というミスがありません。
  </Card>
  <Card title="リアルタイムコラボレーション" icon="group">
    複数人が同じファイルを同時に編集できます。デザイナー・エンジニア・PMが一つのファイルで作業できます。
  </Card>
  <Card title="クロスプラットフォーム" icon="laptop">
    Windows・macOS・Linuxのどの環境でもブラウザから利用できます。デスクトップアプリも提供されています。
  </Card>
  <Card title="無料プランあり" icon="star">
    個人利用や小規模チームは無料プランで十分な機能を使えます。プロジェクト数やファイル数に制限はありますが、学習には十分です。
  </Card>
</CardGrid>

## 主な機能

### デザイン

ベクターグラフィックスを使ったUIデザインが中心機能です。フレーム・コンポーネント・オートレイアウトを組み合わせることで、実際のアプリやWebサイトに近いデザインを作成できます。

- **フレーム**: デバイスサイズに合わせたキャンバス
- **コンポーネント**: 再利用可能なUIパーツ
- **オートレイアウト**: レスポンシブなレイアウト設定

### プロトタイプ

デザインしたフレーム間にリンクを設定することで、クリック可能なプロトタイプを作成できます。実際のアプリのような操作感を再現し、ユーザーテストや関係者へのデモに活用できます。

### コラボレーション

- **コメント機能**: デザイン上に直接コメントを残せる
- **共有リンク**: URLを共有するだけで閲覧・コメントが可能
- **権限管理**: 閲覧のみ・編集可能など細かく設定できる

## 主なユースケース

| ユースケース | 説明 |
|---|---|
| UIデザイン | モバイルアプリ・Webサイトの画面設計 |
| プロトタイピング | クリッカブルなモックアップ作成 |
| デザインシステム | コンポーネントライブラリの管理 |
| ワイヤーフレーム | 低〜高忠実度の画面設計 |
| プレゼン資料 | スライドやビジュアル資料の作成 |

<Aside type="tip">
Figmaは無料アカウントで始められます。まずはブラウザから [figma.com](https://www.figma.com) にアクセスしてアカウントを作成してみましょう。
</Aside>

## FigmaとFigJamの違い

Figmaには「Figma」と「FigJam」の2つのツールがあります。

- **Figma**: UIデザイン・プロトタイプ作成に特化したツール
- **FigJam**: ブレインストーミングやワークショップ向けのオンラインホワイトボード

本ドキュメントでは主に「Figma」（デザインツール）について解説します。
