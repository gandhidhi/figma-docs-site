Figmaは、ブラウザ上で動作するクラウドベースのUIデザインツールです。インストール不要でチームメンバーとリアルタイムに共同作業できる点が最大の特徴です。

主な機能

デザイン

ベクターグラフィックスを使ったUIデザインが中心機能です。フレーム・コンポーネント・オートレイアウトを組み合わせることで、実際のアプリやWebサイトに近いデザインを作成できます。
フレーム: デバイスサイズに合わせたキャンバス
コンポーネント: 再利用可能なUIパーツ
オートレイアウト: レスポンシブなレイアウト設定

プロトタイプ

デザインしたフレーム間にリンクを設定することで、クリック可能なプロトタイプを作成できます。実際のアプリのような操作感を再現し、ユーザーテストや関係者へのデモに活用できます。

コラボレーション
コメント機能: デザイン上に直接コメントを残せる
共有リンク: URLを共有するだけで閲覧・コメントが可能
権限管理: 閲覧のみ・編集可能など細かく設定できる

主なユースケース

| ユースケース | 説明 |
|---|---|
| UIデザイン | モバイルアプリ・Webサイトの画面設計 |
| プロトタイピング | クリッカブルなモックアップ作成 |
| デザインシステム | コンポーネントライブラリの管理 |
| ワイヤーフレーム | 低〜高忠実度の画面設計 |
| プレゼン資料 | スライドやビジュアル資料の作成 |

FigmaとFigJamの違い

Figmaには「Figma」と「FigJam」の2つのツールがあります。
Figma: UIデザイン・プロトタイプ作成に特化したツール
FigJam: ブレインストーミングやワークショップ向けのオンラインホワイトボード
