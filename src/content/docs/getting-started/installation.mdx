Figmaはブラウザから直接使えますが、デスクトップアプリを使うとフォントの利用やオフライン作業が可能になります。

インストール手順

動作環境: Windows 10 以降（64ビット）
  &lt;/TabItem&gt;

動作環境: macOS 11 (Big Sur) 以降。Apple Silicon (M1/M2/M3) にも対応しています。
  &lt;/TabItem&gt;

推奨ブラウザ: Google Chrome（最新版）
  &lt;/TabItem&gt;
&lt;/Tabs&gt;

デスクトップアプリとブラウザ版の比較

| 機能 | デスクトップアプリ | ブラウザ版 |
|---|---|---|
| ローカルフォントの使用 | ✅ 対応 | ⚠️ Font Helper が必要 |
| オフライン作業 | ✅ 対応（制限あり） | ❌ 非対応 |
| 通知機能 | ✅ 対応 | ❌ 非対応 |
| インストール | 必要 | 不要 |
| 起動の手軽さ | ⚠️ アプリ起動が必要 | ✅ URLを開くだけ |

Figma Font Helperのインストール（ブラウザ版のみ）

ブラウザ版でローカルフォントを使用したい場合は、Figma Font Helperをインストールします。
