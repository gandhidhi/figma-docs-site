Figmaのショートカットキーを覚えることで、作業効率が大幅に向上します。よく使う操作から少しずつ覚えていきましょう。

<Aside type="tip">
Figma内で `Ctrl/Cmd + Shift + ?` を押すと、ショートカットキーの一覧パネルが表示されます。
</Aside>

## 選択・移動

| 操作名 | Windows | macOS |
|---|---|---|
| 移動ツール | `V` | `V` |
| 全て選択 | `Ctrl + A` | `Cmd + A` |
| 選択解除 | `Esc` | `Esc` |
| 1px移動 | 矢印キー | 矢印キー |
| 10px移動 | `Shift + 矢印キー` | `Shift + 矢印キー` |
| 前面へ移動 | `Ctrl + ]` | `Cmd + ]` |
| 背面へ移動 | `Ctrl + [` | `Cmd + [` |
| 最前面へ移動 | `Ctrl + Alt + ]` | `Cmd + Option + ]` |
| 最背面へ移動 | `Ctrl + Alt + [` | `Cmd + Option + [` |
| グループ化 | `Ctrl + G` | `Cmd + G` |
| グループ解除 | `Ctrl + Shift + G` | `Cmd + Shift + G` |
| フレーム化 | `Ctrl + Alt + G` | `Cmd + Option + G` |
| 複製 | `Ctrl + D` | `Cmd + D` |
| コピー | `Ctrl + C` | `Cmd + C` |
| 貼り付け | `Ctrl + V` | `Cmd + V` |
| 同じ位置に貼り付け | `Ctrl + Shift + V` | `Cmd + Shift + V` |
| 切り取り | `Ctrl + X` | `Cmd + X` |
| 削除 | `Delete` / `Backspace` | `Delete` / `Backspace` |
