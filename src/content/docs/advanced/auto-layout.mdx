オートレイアウトは、フレーム内の要素を自動的に整列・配置する機能です。コンテンツの増減に応じてフレームが自動的にリサイズされるため、レスポンシブなデザインを効率よく作成できます。

## オートレイアウトとは

通常のフレームでは要素を手動で配置しますが、オートレイアウトを使うと：

- 要素を追加・削除するとフレームが自動的にリサイズされる
- 要素間の間隔を一括で管理できる
- 縦・横方向の並びを簡単に切り替えられる
- CSSのFlexboxに近い概念で、エンジニアとの連携がスムーズになる

## オートレイアウトを適用する

<Steps>
  1. オートレイアウトを適用したいフレームまたはオブジェクトを選択します。

  2. 右パネルの **Auto layout** セクションの **+** ボタンをクリックします。または `Shift + A` のショートカットを使います。

  3. フレームにオートレイアウトが適用され、右パネルに設定オプションが表示されます。
</Steps>

## 主な設定項目

### 方向（Direction）

| 設定 | 説明 |
|---|---|
| 横方向（Horizontal） | 要素を左から右に並べる（行） |
| 縦方向（Vertical） | 要素を上から下に並べる（列） |
| 折り返し（Wrap） | 幅に収まらない場合に折り返す |

### 間隔（Spacing）

- **Gap**: 要素間の間隔
- **Padding**: フレームの内側の余白（上下左右を個別に設定可能）

### 配置（Alignment）

<Tabs>
  <TabItem label="横方向レイアウト">
    - **水平方向**: 左揃え・中央揃え・右揃え・等間隔
    - **垂直方向**: 上揃え・中央揃え・下揃え・ベースライン揃え
  </TabItem>
  <TabItem label="縦方向レイアウト">
    - **水平方向**: 左揃え・中央揃え・右揃え
    - **垂直方向**: 上揃え・中央揃え・下揃え・等間隔
  </TabItem>
</Tabs>

### サイズ調整（Resizing）

各要素のサイズ調整方法を設定できます。

| 設定 | 説明 |
|---|---|
| Fixed | サイズを固定する |
| Hug contents | コンテンツのサイズに合わせて縮小 |
| Fill container | 親フレームの残りスペースを埋める |

<Aside type="tip">
「Fill container」はCSSの `flex: 1` に相当します。複数の要素に設定すると、利用可能なスペースを均等に分割します。
</Aside>

## 実践例：ボタンコンポーネント

テキストの長さに応じて自動的に幅が変わるボタンを作成します。

<Steps>
  1. テキスト「ボタン」を作成します。

  2. テキストを選択して `Shift + A` でオートレイアウトを適用します。

  3. 右パネルで Padding を `12px（上下）× 24px（左右）` に設定します。

  4. フレームの幅を **Hug contents** に設定します。

  5. テキストを変更すると、ボタンの幅が自動的に調整されます。
</Steps>

## 実践例：カードリスト

複数のカードを縦に並べ、カードを追加すると自動的に伸びるリストを作成します。

<Steps>
  1. カードコンポーネントを作成します（例: 幅320px、高さ120px）。

  2. カードを3つ並べて全て選択します。

  3. `Shift + A` でオートレイアウトを適用します。

  4. 方向を **縦方向（Vertical）** に設定します。

  5. Gap を `16px` に設定します。

  6. カードを追加・削除すると、リスト全体の高さが自動的に変わります。
</Steps>

## ネストしたオートレイアウト

オートレイアウトはネスト（入れ子）にできます。複雑なレイアウトも、小さなオートレイアウトを組み合わせることで実現できます。

```
ページレイアウト（縦方向）
├── ヘッダー（横方向）
│   ├── ロゴ
│   └── ナビゲーション（横方向）
├── コンテンツエリア（横方向）
│   ├── サイドバー（縦方向）
│   └── メインコンテンツ（縦方向）
└── フッター（横方向）
```

<Aside type="note">
オートレイアウトはCSSのFlexboxと非常に似た概念です。エンジニアへのデザイン引き渡し時に、オートレイアウトの設定がそのままFlexboxのプロパティとして参照できます。
</Aside>
