Figmaプラグインを使うと、標準機能では対応していない作業を自動化・効率化できます。コミュニティには数千のプラグインが公開されています。

## プラグインをインストールする

<Steps>
  1. Figmaのファイルを開きます。

  2. ツールバーのFigmaメニュー（左上の「F」アイコン）をクリックします。

  3. **Plugins** → **Browse plugins in Community** を選択します。

  4. Figma Communityのプラグインページが開きます。

  5. 使いたいプラグインを検索して見つけます。

  6. プラグインのページで **Install** ボタンをクリックします。

  7. インストール完了後、Figmaメニュー → **Plugins** から使用できます。
</Steps>

## プラグインを実行する

<Steps>
  1. ツールバーのFigmaメニュー → **Plugins** を選択します。

  2. インストール済みのプラグイン一覧が表示されます。

  3. 使いたいプラグインをクリックして起動します。

  4. または `Ctrl/Cmd + /` でクイック検索からプラグインを起動できます。
</Steps>

<Aside type="tip">
よく使うプラグインは右クリックメニューの **Plugins** からも素早くアクセスできます。
</Aside>

## プラグインの管理

### プラグインを削除する

<Steps>
  1. Figmaメニュー → **Plugins** → **Manage plugins** を選択します。

  2. 削除したいプラグインの横にある **...** メニューをクリックします。

  3. **Remove** を選択します。
</Steps>

### プラグインの権限について

プラグインはFigmaファイルへのアクセス権を持ちます。信頼できる開発者のプラグインのみインストールするようにしましょう。

<Aside type="note">
プラグインはFigmaのサンドボックス環境で動作するため、ファイルシステムへの直接アクセスはできません。ただし、ネットワーク通信を行うプラグインもあるため、機密性の高いデザインを扱う場合は注意が必要です。
</Aside>

## プラグインの開発

Figmaプラグインは TypeScript/JavaScript で開発できます。自社のワークフローに特化したプラグインを作成することも可能です。

- [Figma Plugin API ドキュメント](https://www.figma.com/plugin-docs/)
- プラグインはFigma Communityで公開・配布できる
- 社内限定のプライベートプラグインとして配布することも可能
