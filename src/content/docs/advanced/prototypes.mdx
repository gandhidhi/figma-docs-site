プロトタイプ機能を使うと、デザインしたフレーム間にリンクを設定し、実際のアプリのような操作感を再現できます。ユーザーテストや関係者へのデモに活用しましょう。

## プロトタイプモードに切り替える

<Steps>
  1. 右パネル上部の **Prototype** タブをクリックします。

  2. プロトタイプモードに切り替わり、既存のインタラクションが青い矢印で表示されます。
</Steps>

## インタラクションを追加する

<Steps>
  1. 右パネルの **Prototype** タブを開きます。

  2. リンク元にしたいオブジェクト（ボタンなど）を選択します。

  3. オブジェクトの右端に表示される **+** アイコンをクリックするか、オブジェクトの端からドラッグして接続先のフレームに繋げます。

  4. インタラクション設定パネルが表示されます。トリガー・アクション・アニメーションを設定します。

  5. 設定が完了したら右上の **▶ Present** ボタンでプレビューを確認します。
</Steps>

## インタラクションの設定

### トリガー（Trigger）

インタラクションを発動させるユーザー操作を設定します。

| トリガー | 説明 |
|---|---|
| On click | クリック（タップ）したとき |
| On hover | カーソルを乗せたとき |
| On press | 押し続けているとき |
| On drag | ドラッグしたとき |
| While hovering | ホバー中の状態を維持 |
| After delay | 指定時間後に自動で遷移 |
| Key / Gamepad | キーボード入力で発動 |

### アクション（Action）

トリガー発動時の動作を設定します。

| アクション | 説明 |
|---|---|
| Navigate to | 別のフレームに遷移 |
| Open overlay | 現在のフレームの上にオーバーレイ表示 |
| Swap overlay | 表示中のオーバーレイを別のフレームに切り替え |
| Close overlay | オーバーレイを閉じる |
| Back | 前のフレームに戻る |
| Scroll to | 同フレーム内の特定位置にスクロール |
| Open URL | 外部URLを開く |

### アニメーション（Animation）

<Tabs>
  <TabItem label="トランジション">
    フレーム間の遷移アニメーションです。

    - **Instant**: アニメーションなし（即時切り替え）
    - **Dissolve**: フェードイン・フェードアウト
    - **Smart animate**: 同名レイヤーを自動的にアニメーション
    - **Move in / Move out**: スライドイン・スライドアウト
    - **Push**: 押し出すように遷移
    - **Slide in / Slide out**: スライドイン・スライドアウト
  </TabItem>
  <TabItem label="イージング">
    アニメーションの速度変化を設定します。

    - **Linear**: 一定速度
    - **Ease in**: 徐々に加速
    - **Ease out**: 徐々に減速
    - **Ease in and out**: 加速してから減速
    - **Spring**: バネのような動き
    - **Custom**: ベジェ曲線で細かく設定
  </TabItem>
</Tabs>

## Smart Animateの活用

Smart Animateは、2つのフレーム間で同じ名前のレイヤーを自動的にアニメーションさせる機能です。

<Steps>
  1. 遷移元と遷移先のフレームに、同じ名前のレイヤーを配置します（例: 「Card」）。

  2. 遷移元フレームでは「Card」を小さく配置し、遷移先では大きく配置します。

  3. インタラクションのアニメーションを **Smart animate** に設定します。

  4. プレビューすると、カードが滑らかに拡大するアニメーションが再生されます。
</Steps>

<Aside type="tip">
Smart Animateはモーションデザインの知識がなくても、プロフェッショナルなアニメーションを作成できる強力な機能です。
</Aside>

## プロトタイプのプレビューと共有

### プレビューする

<Steps>
  1. 右上の **▶ Present** ボタンをクリックします。

  2. プレビューウィンドウが開き、インタラクションを実際に操作できます。

  3. 左上のデバイスアイコンからプレビューするデバイスを変更できます。
</Steps>

### プロトタイプを共有する

<Steps>
  1. プレビューウィンドウの右上にある **Share prototype** ボタンをクリックします。

  2. 共有リンクが生成されます。

  3. リンクをコピーして関係者に送ります。受け取った人はFigmaアカウントなしでプロトタイプを閲覧できます。
</Steps>

<Aside type="note">
共有リンクのアクセス権限は「リンクを知っている全員」または「特定のユーザーのみ」に設定できます。機密性の高いデザインは権限設定に注意してください。
</Aside>
