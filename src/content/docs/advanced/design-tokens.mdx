デザイントークンとスタイルを活用することで、デザインの一貫性を保ちながら、変更を効率的に管理できます。

## スタイルとは

スタイルは、色・テキスト・エフェクト・グリッドなどのデザイン値を名前付きで保存する機能です。スタイルを変更すると、そのスタイルを使用している全ての要素に変更が反映されます。

## カラースタイルの管理

### カラースタイルを作成する

<Steps>
  1. 色を設定したいオブジェクトを選択します。

  2. 右パネルの「塗り」セクションで色を設定します。

  3. 色の横にある **4つの点のアイコン** をクリックします。

  4. **+** ボタンをクリックして新しいスタイルを作成します。

  5. スタイル名を入力します（例: `Primary/500`、`Neutral/100`）。

  6. **Create style** をクリックして保存します。
</Steps>

### カラースタイルの命名規則

```
カテゴリ/色名/トーン
例:
- Primary/500（メインカラー）
- Primary/100（薄いメインカラー）
- Neutral/900（ほぼ黒）
- Neutral/100（ほぼ白）
- Semantic/Success（成功を表す緑）
- Semantic/Error（エラーを表す赤）
- Semantic/Warning（警告を表す黄）
```

<Aside type="tip">
カラースタイルはブランドカラーやUIの状態（成功・エラー・警告）を定義するのに最適です。デザインシステムの基盤となります。
</Aside>

## テキストスタイルの管理

### テキストスタイルを作成する

<Steps>
  1. テキストを選択します。

  2. フォント・サイズ・行間などを設定します。

  3. 右パネルの「テキスト」セクション横の **4つの点のアイコン** をクリックします。

  4. **+** ボタンをクリックして新しいスタイルを作成します。

  5. スタイル名を入力します（例: `Heading/H1`、`Body/Regular`）。
</Steps>

### テキストスタイルの命名規則

```
カテゴリ/バリアント
例:
- Heading/H1（見出し1）
- Heading/H2（見出し2）
- Body/Regular（本文・通常）
- Body/Bold（本文・太字）
- Caption/Regular（キャプション）
- Label/Small（ラベル・小）
```

## エフェクトスタイルとグリッドスタイル

<Tabs>
  <TabItem label="エフェクトスタイル">
    ドロップシャドウやぼかしをスタイルとして保存できます。

    - カードの影（`Shadow/Card`）
    - モーダルの影（`Shadow/Modal`）
    - ホバー時の影（`Shadow/Hover`）
  </TabItem>
  <TabItem label="グリッドスタイル">
    レイアウトグリッドをスタイルとして保存できます。

    - モバイル用グリッド（4カラム）
    - タブレット用グリッド（8カラム）
    - デスクトップ用グリッド（12カラム）
  </TabItem>
</Tabs>

## デザイントークン

デザイントークンは、スタイルをさらに抽象化した概念です。色・サイズ・間隔などのデザイン値を、プラットフォームに依存しない形式（JSON）で管理します。

### デザイントークンの例

```json
{
  "color": {
    "primary": {
      "500": { "value": "#3B82F6" },
      "100": { "value": "#DBEAFE" }
    },
    "semantic": {
      "success": { "value": "{color.green.500}" },
      "error": { "value": "{color.red.500}" }
    }
  },
  "spacing": {
    "xs": { "value": "4px" },
    "sm": { "value": "8px" },
    "md": { "value": "16px" },
    "lg": { "value": "24px" }
  }
}
```

Figmaの標準機能ではデザイントークンのJSON出力はサポートされていませんが、プラグインを使って実現できます。

## スタイルをライブラリとして共有する

チーム全体でスタイルを共有するには、ライブラリとして公開します。

<Steps>
  1. スタイルを定義したファイルを開きます。

  2. ツールバーのFigmaメニュー → **Libraries** を選択します。

  3. **Publish** をクリックしてライブラリを公開します。

  4. 他のファイルでアセットパネルからこのライブラリのスタイルを使用できます。
</Steps>

<Aside type="note">
スタイルを変更してライブラリを更新すると、そのスタイルを使用している他のファイルに更新通知が届きます。ファイルを開いた際に「Update available」と表示されたら、更新を適用してください。
</Aside>
