Figmaはリアルタイムコラボレーションを前提に設計されています。ファイルの共有設定とコメント機能を活用して、チームでの作業を効率化しましょう。

## ファイルを共有する

<Steps>
  1. 共有したいファイルを開きます。

  2. 右上の **Share** ボタンをクリックします。

  3. 共有設定パネルが開きます。
</Steps>

## 権限の種類

Figmaでは共有相手に応じて権限を設定できます。

| 権限 | できること |
|---|---|
| Can edit | ファイルの編集・コメント・閲覧 |
| Can view | 閲覧・コメントのみ（編集不可） |
| Can view prototypes | プロトタイプの閲覧のみ |

<Aside type="tip">
エンジニアやクライアントには「Can view」権限を付与するのが一般的です。誤って編集されるリスクを防げます。
</Aside>

## 共有方法

### メールアドレスで招待する

<Steps>
  1. 共有パネルの入力欄にメールアドレスを入力します。

  2. 権限（Can edit / Can view）を選択します。

  3. 必要に応じてメッセージを入力します。

  4. **Send invite** をクリックします。

  5. 招待されたユーザーにメールが届き、承認するとファイルにアクセスできます。
</Steps>

### リンクで共有する

<Steps>
  1. 共有パネルの **Copy link** ボタンをクリックします。

  2. リンクのアクセス設定を選択します：
     - **Only people invited to this file**: 招待済みのユーザーのみ
     - **Anyone with the link**: リンクを知っている全員

  3. コピーしたリンクをSlackやメールで送ります。
</Steps>

## チームメンバーとのリアルタイム共同作業

同じファイルを複数人で同時に開くと、お互いのカーソルがリアルタイムで表示されます。

<CardGrid>
  <Card title="カーソルの表示" icon="eye">
    他のメンバーのカーソルが名前付きで表示されます。誰がどこを作業しているか一目でわかります。
  </Card>
  <Card title="フォロー機能" icon="group">
    他のメンバーのアバターをクリックすると、そのメンバーの画面をリアルタイムで追跡できます。レビューやペアデザインに便利です。
  </Card>
</CardGrid>

## コメント機能

### コメントを追加する

<Steps>
  1. ツールバーのコメントツール（`C` キー）を選択します。

  2. コメントを追加したい場所をクリックします。

  3. コメント入力欄が表示されます。テキストを入力します。

  4. `@メンション` で特定のメンバーに通知を送れます。

  5. `Enter` または **Post** ボタンでコメントを投稿します。
</Steps>

### コメントに返信する

<Steps>
  1. キャンバス上のコメントアイコンをクリックします。

  2. コメントパネルが開きます。

  3. 返信欄にテキストを入力して **Reply** をクリックします。
</Steps>

### コメントを解決する

対応が完了したコメントは「解決済み」にしてスレッドを整理できます。

<Steps>
  1. コメントパネルを開きます。

  2. 右上の **✓ Resolve** ボタンをクリックします。

  3. コメントが解決済みとしてマークされ、キャンバスから非表示になります。

  4. 解決済みコメントは **Resolved** フィルターで確認できます。
</Steps>

<Aside type="note">
コメントはデザインレビューのフローに組み込むと効果的です。デザイナーがデザインを完成させたら共有し、レビュアーがコメントで指摘、デザイナーが修正してコメントを解決、というサイクルを回しましょう。
</Aside>

## 観察者モード（Observation Mode）

プレゼンテーションやデザインレビュー時に便利な機能です。

<Steps>
  1. ファイルを開いた状態で、右上のメンバーアバターをクリックします。

  2. **Spotlight me** を選択すると、他のメンバーが自分の画面を追跡できる状態になります。

  3. 他のメンバーのアバターをクリックして **Follow** を選択すると、そのメンバーの画面を追跡できます。
</Steps>
