コンポーネントはFigmaの最も重要な機能の一つです。ボタンやカードなどのUIパーツをコンポーネント化することで、デザインの一貫性を保ちながら効率的に作業できます。

## コンポーネントとインスタンスの関係

<CardGrid>
  <Card title="メインコンポーネント" icon="star">
    コンポーネントの「原本」です。変更するとすべてのインスタンスに反映されます。レイヤーパネルでは紫色の4つの菱形アイコンで表示されます。
  </Card>
  <Card title="インスタンス" icon="copy">
    メインコンポーネントから作られた「コピー」です。メインの変更を継承しつつ、個別にプロパティを上書きできます。
  </Card>
</CardGrid>

## コンポーネントを作成する

<Steps>
  1. コンポーネント化したいオブジェクト（またはグループ）を選択します。

  2. 右クリックして **Create component** を選択します。または `Ctrl/Cmd + Alt + K` のショートカットを使います。

  3. レイヤーパネルのアイコンが紫色の菱形に変わり、コンポーネントが作成されます。

  4. コンポーネント名をダブルクリックして分かりやすい名前に変更します。
</Steps>

<Aside type="tip">
コンポーネント名に `/` を使うとグループ化されます。例えば `Button/Primary`、`Button/Secondary` と命名すると、アセットパネルで「Button」グループとして整理されます。
</Aside>

## インスタンスを作成する

<Steps>
  1. 左パネルの **アセットタブ**（Shift + I）を開きます。

  2. 使いたいコンポーネントを検索または一覧から見つけます。

  3. コンポーネントをキャンバスにドラッグ＆ドロップします。

  4. または、メインコンポーネントを選択して `Ctrl/Cmd + D` で複製するとインスタンスが作成されます。
</Steps>

## インスタンスのプロパティを上書きする

インスタンスはメインコンポーネントのスタイルを継承しますが、個別に変更（上書き）できます。

### 上書きできるプロパティ

- **テキスト内容**: ボタンのラベルなど
- **画像**: アバターやサムネイル画像
- **色**: 特定のインスタンスだけ色を変える
- **表示・非表示**: 特定の要素を隠す

### 上書きをリセットする

インスタンスを右クリックして **Reset all overrides** を選択すると、メインコンポーネントの状態に戻せます。

## バリアントの活用

バリアントを使うと、1つのコンポーネントで複数のバリエーション（状態）を管理できます。

### バリアントの例

```
Button コンポーネント
├── Type: Primary / Secondary / Danger
├── Size: Small / Medium / Large
└── State: Default / Hover / Disabled
```

### バリアントを作成する

<Steps>
  1. メインコンポーネントを選択します。

  2. 右パネルの **Add variant** ボタンをクリックします。

  3. 新しいバリアントが追加されます。プロパティ名と値を設定します。

  4. 各バリアントのデザインを個別に編集します。
</Steps>

<Aside type="note">
バリアントはFigmaの有料プランでより高度な機能が使えます。無料プランでも基本的なバリアントは作成できます。
</Aside>

## コンポーネントのベストプラクティス

### 命名規則

```
カテゴリ/コンポーネント名/バリアント
例:
- Button/Primary
- Button/Secondary
- Form/Input/Default
- Form/Input/Error
- Navigation/Tab/Active
```

### コンポーネントの整理

- **専用ページを作る**: デザインファイル内に「Components」ページを作り、全コンポーネントをまとめる
- **カテゴリ別にフレームで区切る**: ボタン・フォーム・カードなどカテゴリごとにフレームで整理
- **説明を追加する**: コンポーネントを選択して右パネルの「Description」に使い方を記述

### デザインシステムへの発展

コンポーネントが増えてきたら、別のファイルを「コンポーネントライブラリ」として公開し、チーム全体で共有できます。

<Steps>
  1. コンポーネントをまとめたファイルを開きます。

  2. ツールバーのFigmaメニュー → **Libraries** を選択します。

  3. **Publish** をクリックしてライブラリを公開します。

  4. 他のファイルでアセットパネルからこのライブラリのコンポーネントを使用できます。
</Steps>
