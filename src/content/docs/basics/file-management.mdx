Figmaではファイルを「プロジェクト」という単位でまとめて管理します。チームでの作業では、プロジェクトを適切に整理することが重要です。

## ファイルとプロジェクトの構造

```
チーム（Team）
└── プロジェクト（Project）
    ├── デザインファイル（.fig）
    ├── デザインファイル（.fig）
    └── FigJamファイル（.jam）
```

- **チーム**: 組織やグループ単位の最上位コンテナ
- **プロジェクト**: 関連するファイルをまとめるフォルダ
- **ファイル**: 実際のデザインが入る単位

## 新しいファイルを作成する

<Steps>
  1. Figmaのホーム画面（figma.com）を開きます。

  2. 左サイドバーからファイルを作成したいプロジェクトを選択します。

  3. 右上の **+ New design file** ボタンをクリックします。

  4. 新しいデザインファイルが開きます。ファイル名はツールバー中央に表示されており、クリックして変更できます。
</Steps>

<Aside type="tip">
ドラフト（Drafts）はプロジェクトに属さない個人作業用のスペースです。試作や個人的なメモはドラフトに保存し、チームと共有する準備ができたらプロジェクトに移動しましょう。
</Aside>

## ファイルを複製する

既存のファイルをテンプレートとして使いたい場合は複製が便利です。

<Steps>
  1. ホーム画面でファイルのサムネイルを右クリックします。

  2. コンテキストメニューから **Duplicate** を選択します。

  3. 同じプロジェクト内に「ファイル名 (Copy)」という名前で複製されます。

  4. 複製されたファイルをクリックして開き、名前を変更します。
</Steps>

## ファイルを移動する

<Steps>
  1. ホーム画面でファイルのサムネイルを右クリックします。

  2. **Move to project** を選択します。

  3. 移動先のプロジェクトを選択して **Move** をクリックします。
</Steps>

## プロジェクトを作成・管理する

### 新しいプロジェクトを作成する

<Steps>
  1. ホーム画面の左サイドバーでチーム名の横にある **+** アイコンをクリックします。

  2. **New project** を選択します。

  3. プロジェクト名を入力して **Create project** をクリックします。
</Steps>

### プロジェクトの整理のコツ

<CardGrid>
  <Card title="プロダクト別に分ける" icon="document">
    「モバイルアプリ」「Webサイト」「マーケティング」など、プロダクトや用途ごとにプロジェクトを分けると管理しやすくなります。
  </Card>
  <Card title="命名規則を統一する" icon="pencil">
    「[プロダクト名] - [フェーズ]」のような命名規則を決めておくと、チームメンバーが迷わずファイルを見つけられます。
  </Card>
</CardGrid>

## ファイルのバージョン管理

Figmaはファイルの変更履歴を自動的に保存します。

### バージョン履歴を確認する

<Steps>
  1. ファイルを開いた状態でツールバーのファイル名をクリックします。

  2. **Show version history** を選択します。

  3. 右パネルに変更履歴が時系列で表示されます。

  4. 任意のバージョンをクリックするとその時点のデザインをプレビューできます。

  5. 過去のバージョンに戻したい場合は、そのバージョンを右クリックして **Restore this version** を選択します。
</Steps>

### 手動でバージョンを保存する

重要な変更を加えた際は手動でバージョンを保存しておくと便利です。

- `Ctrl/Cmd + Alt + S` でバージョンを保存
- バージョン名と説明を入力して保存できる

## ファイルのエクスポートとインポート

### ファイルをエクスポートする

<Steps>
  1. ツールバーのFigmaメニュー（左上の「F」アイコン）をクリックします。

  2. **File** → **Save local copy** を選択します。

  3. `.fig` 形式でローカルに保存されます。
</Steps>

### ファイルをインポートする

<Steps>
  1. ホーム画面の右上にある **Import** ボタンをクリックします。

  2. `.fig` ファイルまたは `.sketch` ファイルを選択します。

  3. インポートが完了するとドラフトに追加されます。
</Steps>

<Aside type="note">
Sketchファイル（`.sketch`）のインポートはサポートされていますが、一部のエフェクトやフォントが正しく変換されない場合があります。インポート後に確認することをお勧めします。
</Aside>
