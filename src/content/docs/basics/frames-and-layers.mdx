フレームはFigmaデザインの基本単位です。レイヤーパネルと組み合わせることで、複雑なデザインも整理して管理できます。

## フレームとは

フレームは、デザインを配置するための「キャンバス内のキャンバス」です。スマートフォンやPCの画面サイズに合わせたフレームを作成し、その中にUIパーツを配置します。

- **デバイスフレーム**: iPhone・Android・デスクトップなどのプリセットサイズ
- **カスタムフレーム**: 任意のサイズで作成
- **ネストフレーム**: フレームの中にフレームを入れることも可能

<Aside type="tip">
Figmaでは「アートボード」という概念はなく、すべて「フレーム」として扱います。他のデザインツールからの移行時は注意してください。
</Aside>

## フレームを作成する

<Steps>
  1. ツールバーの **フレームツール**（`F` キー）を選択します。

  2. キャンバス上でドラッグしてフレームを描きます。または、右パネルのプリセットからデバイスサイズを選択します。

  3. フレームが作成されると、レイヤーパネルにフレームアイコン（#）付きで表示されます。

  4. フレームを選択した状態で右パネルの「W」「H」に数値を入力してサイズを変更できます。
</Steps>

### プリセットサイズの一覧

| カテゴリ | サイズ例 |
|---|---|
| Phone | iPhone 14 (390×844)、Android Large (360×800) |
| Tablet | iPad (768×1024)、iPad Pro 12.9" (1024×1366) |
| Desktop | Desktop (1440×1024)、MacBook Pro (1512×982) |
| Presentation | Slide 16:9 (1920×1080) |
| Social Media | Instagram Post (1080×1080) |

## レイヤーパネルの操作

### レイヤーの基本操作

<Tabs>
  <TabItem label="選択">
    - レイヤーパネルのレイヤー名をクリックすると選択できます
    - `Ctrl/Cmd` を押しながらクリックで複数選択
    - `Shift` を押しながらクリックで範囲選択
  </TabItem>
  <TabItem label="並び替え">
    - レイヤーをドラッグ＆ドロップで順序を変更できます
    - 上にあるレイヤーほどキャンバス上で前面に表示されます
    - `Ctrl/Cmd + ]` で前面へ、`Ctrl/Cmd + [` で背面へ移動
  </TabItem>
  <TabItem label="グループ化">
    - 複数のレイヤーを選択して `Ctrl/Cmd + G` でグループ化
    - グループ名をダブルクリックして名前を変更
    - `Ctrl/Cmd + Shift + G` でグループを解除
  </TabItem>
  <TabItem label="表示・ロック">
    - レイヤー名にカーソルを合わせると目のアイコンが表示される
    - 目のアイコンをクリックでレイヤーの表示・非表示を切り替え
    - 鍵のアイコンをクリックでレイヤーをロック（誤操作防止）
  </TabItem>
</Tabs>

### レイヤーの命名規則

チームで作業する際は、レイヤーに分かりやすい名前をつけることが重要です。

```
推奨命名例:
├── Header
│   ├── Logo
│   ├── Navigation
│   └── CTA Button
├── Hero Section
│   ├── Background Image
│   ├── Headline
│   └── Subtext
└── Footer
```

<Aside type="tip">
レイヤー名に `/` を使うとFigmaがグループとして認識します。例えば `Icon/Home`、`Icon/Settings` と命名すると、アセットパネルで「Icon」グループとして整理されます。
</Aside>

## フレームのクリッピング

フレームはデフォルトで「クリップコンテンツ」が有効になっており、フレームの外にはみ出た要素は表示されません。

- 右パネルの **Clip content** チェックボックスで切り替え可能
- ドロップダウンメニューやツールチップなど、フレームからはみ出す要素を表示したい場合はオフにする

## フレームのリサイズ

フレームのサイズを変更する際、中のコンテンツをどう扱うかを設定できます。

| 設定 | 動作 |
|---|---|
| Scale | フレームに合わせてコンテンツも拡縮 |
| Hug contents | コンテンツのサイズに合わせてフレームが縮小 |
| Fixed | フレームサイズを固定（コンテンツは変化しない） |

<Aside type="note">
オートレイアウトを使用したフレームでは、リサイズの動作がより細かく制御できます。詳しくは[オートレイアウト](/advanced/auto-layout)のページを参照してください。
</Aside>
