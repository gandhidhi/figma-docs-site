Figmaのエディタ画面は大きく5つのエリアに分かれています。各エリアの名称と役割を把握することで、操作をスムーズに行えます。

画面全体のレイアウト

`
┌─────────────────────────────────────────────────────┐
│                   ツールバー（上部）                    │
├──────────┬──────────────────────────┬───────────────┤
│          │                          │               │
│  レイヤー  │       キャンバス           │  デザイン      │
│  パネル   │       （中央）             │  パネル        │
│  （左）   │                          │  （右）        │
│          │                          │               │
└──────────┴──────────────────────────┴───────────────┘
`

各エリアの説明

ツールバー（上部）

画面上部に固定されているバーです。

| 要素 | 説明 |
|---|---|
| Figmaメニュー（左端の「F」アイコン） | ファイル操作・設定・プラグインなどのメニュー |
| ツール選択 | 移動・フレーム・図形・テキスト・ペンなどのツール |
| ファイル名 | クリックで編集可能。ファイルの保存状態も表示 |
| 共有ボタン | ファイルの共有設定を開く |
| プレビューボタン | プロトタイプのプレビューを開始 |
| ズーム表示 | 現在のズーム倍率。クリックで変更可能 |

レイヤーパネル（左側）

画面左側のパネルです。上部のタブで切り替えられます。
レイヤータブ: ファイル内の全レイヤー・フレームの階層構造を表示
アセットタブ: コンポーネント・スタイル・プラグインのアセットを表示

キャンバス（中央）

デザインを実際に作業するメインエリアです。
ホワイトスペース: フレームの外側の無限に広がる作業領域
フレーム: デバイスサイズを表す枠。実際のデザインはここに配置
ズーム操作: Ctrl/Cmd + スクロール でズームイン・アウト
パン操作: スペースキーを押しながらドラッグで移動

デザインパネル（右側）

選択中のオブジェクトのプロパティを表示・編集するパネルです。

| タブ | 内容 |
|---|---|
| デザイン | 位置・サイズ・色・角丸・エフェクトなど |
| プロトタイプ | インタラクション・アニメーション設定 |
| インスペクト | エンジニア向けのCSS値・コード表示 |

ステータスバー（下部）

画面下部に表示される情報バーです。
現在のズーム倍率
選択中のオブジェクト情報
コラボレーター（同時編集中のユーザー）のアバター

よく使うショートカット

| 操作 | Windows | macOS |
|---|---|---|
| ズームリセット | Shift + 0 | Shift + 0 |
| 全体表示 | Shift + 1 | Shift + 1 |
| 選択範囲に合わせる | Shift + 2 | Shift + 2 |
| ツールバー非表示 | Ctrl + \ | Cmd + \ |
