Figmaでデザインを作成する際の基本要素である図形・テキスト・画像の操作方法を解説します。

図形の配置

基本図形ツール

ツールバーの図形ツールから各種図形を選択できます。

| ツール | ショートカット | 説明 |
|---|---|---|
| 長方形 | R | 四角形・正方形の描画 |
| 楕円 | O | 円・楕円の描画 |
| 直線 | L | 直線の描画 |
| 矢印 | Shift + L | 矢印付き直線の描画 |
| 多角形 | なし | 三角形・多角形の描画 |
| 星形 | なし | 星形の描画 |

図形を描く

図形のプロパティ

右パネルで以下のプロパティを設定できます。
位置（X, Y）: キャンバス上の座標
サイズ（W, H）: 幅と高さ
角丸（Corner radius）: 角を丸くする半径
塗り（Fill）: 単色・グラデーション・画像で塗りつぶし
線（Stroke）: 枠線の色・太さ・スタイル
エフェクト: ドロップシャドウ・ぼかしなど

オブジェクトの整列と分布

複数のオブジェクトを選択すると、右パネルに整列ツールが表示されます。

| アイコン | 機能 |
|---|---|
| 左揃え | 選択オブジェクトを左端に揃える |
| 水平中央揃え | 水平方向の中央に揃える |
| 右揃え | 右端に揃える |
| 上揃え | 上端に揃える |
| 垂直中央揃え | 垂直方向の中央に揃える |
| 下揃え | 下端に揃える |
| 水平等間隔 | 水平方向に等間隔で分布 |
| 垂直等間隔 | 垂直方向に等間隔で分布 |
