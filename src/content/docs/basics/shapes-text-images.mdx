Figmaでデザインを作成する際の基本要素である図形・テキスト・画像の操作方法を解説します。

## 図形の配置

### 基本図形ツール

ツールバーの図形ツールから各種図形を選択できます。

| ツール | ショートカット | 説明 |
|---|---|---|
| 長方形 | `R` | 四角形・正方形の描画 |
| 楕円 | `O` | 円・楕円の描画 |
| 直線 | `L` | 直線の描画 |
| 矢印 | `Shift + L` | 矢印付き直線の描画 |
| 多角形 | なし | 三角形・多角形の描画 |
| 星形 | なし | 星形の描画 |

### 図形を描く

<Steps>
  1. ツールバーから描きたい図形ツールを選択します（またはショートカットキーを押します）。

  2. キャンバス上でドラッグして図形を描きます。

  3. `Shift` を押しながらドラッグすると正方形・正円など縦横比が固定されます。

  4. 図形を選択した状態で右パネルからサイズ・色・角丸などを調整します。
</Steps>

### 図形のプロパティ

右パネルで以下のプロパティを設定できます。

- **位置（X, Y）**: キャンバス上の座標
- **サイズ（W, H）**: 幅と高さ
- **角丸（Corner radius）**: 角を丸くする半径
- **塗り（Fill）**: 単色・グラデーション・画像で塗りつぶし
- **線（Stroke）**: 枠線の色・太さ・スタイル
- **エフェクト**: ドロップシャドウ・ぼかしなど

<Aside type="tip">
`Ctrl/Cmd + D` で選択中のオブジェクトを複製できます。同じ要素を繰り返し配置する際に便利です。
</Aside>

## テキストの配置

### テキストを追加する

<Steps>
  1. ツールバーの **テキストツール**（`T` キー）を選択します。

  2. キャンバス上でクリックすると「自動サイズ」のテキストボックスが作成されます。ドラッグすると固定幅のテキストボックスが作成されます。

  3. テキストを入力します。

  4. `Esc` キーを押すとテキスト編集モードを終了し、オブジェクト選択モードに戻ります。
</Steps>

### テキストのプロパティ

<Tabs>
  <TabItem label="フォント設定">
    右パネルの「テキスト」セクションで設定できます。

    - **フォントファミリー**: フォントの種類（Noto Sans JP、Robotoなど）
    - **フォントウェイト**: Regular・Medium・Bold など
    - **フォントサイズ**: ピクセル単位で指定
    - **行間（Line height）**: 行と行の間隔
    - **字間（Letter spacing）**: 文字と文字の間隔
  </TabItem>
  <TabItem label="テキスト配置">
    - **水平方向**: 左揃え・中央揃え・右揃え・両端揃え
    - **垂直方向**: 上揃え・中央揃え・下揃え
    - **テキストの折り返し**: 自動・固定幅・固定高さ
  </TabItem>
  <TabItem label="テキストスタイル">
    よく使うテキストスタイルは「スタイル」として保存できます。

    1. テキストを選択して右パネルの「テキスト」横の **+** をクリック
    2. スタイル名を入力して保存
    3. 他のテキストに同じスタイルを適用できる
  </TabItem>
</Tabs>

## 画像の配置

### 画像をインポートする

<Tabs>
  <TabItem label="ドラッグ＆ドロップ">
    <Steps>
      1. ローカルの画像ファイル（PNG・JPG・SVG・GIF・WebP）をFigmaのキャンバスにドラッグ＆ドロップします。

      2. 画像がキャンバスに配置されます。
    </Steps>
  </TabItem>
  <TabItem label="メニューから挿入">
    <Steps>
      1. ツールバーのFigmaメニュー → **File** → **Place image** を選択します。

      2. ファイル選択ダイアログから画像を選択します。

      3. キャンバス上でクリックまたはドラッグして配置します。
    </Steps>
  </TabItem>
  <TabItem label="図形に画像を適用">
    <Steps>
      1. 図形を選択します。

      2. 右パネルの「塗り」セクションで塗りの種類を **Image** に変更します。

      3. 画像ファイルを選択します。

      4. 画像のフィット方法（Fill・Fit・Crop・Tile）を選択します。
    </Steps>
  </TabItem>
</Tabs>

### 画像のトリミング

<Steps>
  1. 画像を選択します。

  2. 右パネルの塗りセクションで **Crop** を選択します。

  3. 画像の端をドラッグしてトリミング範囲を調整します。

  4. `Enter` キーを押してトリミングを確定します。
</Steps>

## オブジェクトの整列と分布

複数のオブジェクトを選択すると、右パネルに整列ツールが表示されます。

| アイコン | 機能 |
|---|---|
| 左揃え | 選択オブジェクトを左端に揃える |
| 水平中央揃え | 水平方向の中央に揃える |
| 右揃え | 右端に揃える |
| 上揃え | 上端に揃える |
| 垂直中央揃え | 垂直方向の中央に揃える |
| 下揃え | 下端に揃える |
| 水平等間隔 | 水平方向に等間隔で分布 |
| 垂直等間隔 | 垂直方向に等間隔で分布 |

<Aside type="tip">
`Alt/Option` を押しながらオブジェクトにカーソルを合わせると、他のオブジェクトとの距離が表示されます。正確な間隔でレイアウトを組む際に便利です。
</Aside>
